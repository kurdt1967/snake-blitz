Snake Game

How to Play

In Manual Mode, you control the snake using the arrow keys to eat food while avoiding obstacles. The snake will grow longer each time it eats food, and the game speed will increase as the level goes up. If the snake crashes into the wall or its own body, the game will be over.

There is also a booster functionality, if you hold down the spacebar, the snake will go a bit faster


In AI Mode, the AI snake will use a pathfinding greedy algorithm to move the snake towards the food.
